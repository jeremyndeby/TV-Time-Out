# TV Time Out

A Chrome extension to export your TV Time library fast.

## Features

- Exports your full watch history — series, movies, and episodes
- Includes watch dates, episode status, and metadata
- 100% client-side — no data leaves your browser
- Works for large libraries

## Export format

Two JSON files:
- `tvtime-series-{date}.json` — all followed shows with seasons and episodes
- `tvtime-movies-{date}.json` — all movies with watch status

## Installation (Developer Mode)

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer Mode** (top right)
4. Click **Load unpacked** and select this folder

## Usage

1. Open [app.tvtime.com](https://app.tvtime.com) and log in
2. Click the TV Time Out icon in your toolbar
3. Click **Export my data**
4. Your files will download automatically

## Privacy

TV Time Out does not collect, store, or transmit any data.
All processing happens locally in your browser.
